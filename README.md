PIZZA_SALES

Overview:
PIZZA_SALES is a MySQL-based project analyzing pizza sales data. The project utilizes structured queries to retrieve meaningful insights from a pizza store dataset.

Features:
Perform advanced SQL operations using DML, DDL, DCL, and joins.
Retrieve data across schemas for in-depth analysis.
Generate business insights, such as revenue and customer preferences.

Database Details:
Database Name: Pizza hut

Tables:
order_details
orders
pizza_types
pizzas

Key Queries:
Total revenue generated from pizza sales.
Highest-priced pizza identification.
Top 5 most-ordered pizza types and their quantities.
Distribution of orders by hour of the day.
Category-wise pizza distribution.
Average number of pizzas ordered per day.

Insights:
Determined the total number of orders placed.
Identified the 3 topmost ordered pizza types by revenue.
Calculated hourly and category-based distribution of orders.

How to Run:
Import the provided dataset into MySQL.
Execute SQL queries to retrieve insights.
Analyze results to understand trends and metrics.
